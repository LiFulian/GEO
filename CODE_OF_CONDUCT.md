# 社区行为准则

## 我们的承诺

为了营造一个开放、友好的环境，我们作为贡献者和维护者承诺：无论年龄、体型、残疾、族裔、性别认同与表达、经验水平、国籍、外貌、种族、宗教或性取向如何，参与本项目的每个人都享有无骚扰的体验。

## 我们的准则

**有益的行为**包括：

- 🤝 使用友好和包容的语言
- 🎯 尊重不同的观点和经验
- 📝 优雅地接受建设性批评
- 🌱 关注对社区最有利的事情
- 💚 对其他社区成员表示同理心

**不可接受的行为**包括：

- ❌ 使用性化的语言、图像或挑逗性举动
- ❌ 恶意评论、侮辱/贬损言论、攻击性人身或政治攻击
- ❌ 公开或私下骚扰
- ❌ 未经明确许可发布他人的私人信息（"doxing"）
- ❌ 其他在专业场合可被合理认为不当的行为

## 适用范围

本准则适用于本项目的所有空间——GitHub issue / PR / Discussion、官方群聊、以及任何代表项目发言的场合。

## 执行

如有滥用、骚扰或其他不可接受的行为，请联系项目维护团队：**maintainers@geo-studio.local**（占位，请替换为实际邮箱）。

所有投诉都会被审查和调查，并导致被认为必要且适当的回应。项目团队有义务对事件的报告者保密。
